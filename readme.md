## What pages are included

- Home page (landing)
- Showcase (some fake project examples)
- Products list (browse items)
- Product detail modal (click "Buy Now")
- Register page
- Login page
- Checkout page (with fake verification modal)
- Profile page (with fake purchases & settings)

## Tech stack (very basic)

- HTML
- CSS (just plain CSS, no framework)
- Vanilla JavaScript
- Some Unsplash images for placeholders
- postgresql
- express.js
- bun

## How it looks

Mostly indigo/purple colors (#6366f1), rounded corners, some hover effects, mobile responsive.

## How to run it locally
1. bun install
2. bun run dev
3. open localhost:3100/pages/ in browser

## What works ?

- Looks okay on desktop & mobile  
- modals (product detail, payment verification)  
- tabs on profile  
- Password show/hide on login/register (Frontend and backend)

- No real payment  
- saving data (data saved in database or local storage)
